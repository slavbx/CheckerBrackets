Spring Boot-приложение для проверки скобок в тексте
Среда разработки: IntelliJ IDEA v2022  
Версия JDK: 17  
   
Описание:
Принимает POST-запросы на http://localhost:8080/api/checkBrackets  
в виде JSON:  
> {  
"text": "Вчера я отправился в поход в лес (это мое любимое место для отдыха) вместе с друзьями. Мы выбрали маршрут, который проходил через горные потоки и поля (для разнообразия).  
}

Пример ответа для корректного текста:

> {  
"isCorrect" : true  
}

Пример ответа для некорректного текста:

> {  
"isCorrect" : false  
}
